Adidas Sales Dashboard
Interactive Data Analytics and Business Insights Project

Overview
This project focuses on analyzing Adidas sales data using Excel to build an interactive dashboard. The dashboard provides clear visual insights into sales performance, product trends, and regional distribution to support data-driven business decisions.

Objectives

* Analyze sales performance data for Adidas products
* Identify trends in revenue and product demand
* Visualize key performance indicators (KPIs)
* Build an interactive and user-friendly dashboard
* Support business decision-making through data insights

Tools and Technologies

* Microsoft Excel
* Data Visualization
* Pivot Tables
* Charts and Graphs
* Data Cleaning and Processing

Key Features

* Interactive sales dashboard
* Regional sales analysis
* Product performance tracking
* Monthly and yearly sales trends
* KPI visualization (Revenue, Units Sold, Profit)

Dashboard Components

* Sales overview section
* Product performance analysis
* Regional breakdown
* Time-based trends
* Interactive filters and slicers
